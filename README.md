# 🌿 Agricultura Sustentável - Agrinho 2026

Projeto desenvolvido para a categoria Front-End do Agrinho.

## Tecnologias
- HTML5
- CSS3
- JavaScript

## Tema
Agricultura e Sustentabilidade

## Objetivo
Mostrar a importância da sustentabilidade no campo e da preservação ambiental.
